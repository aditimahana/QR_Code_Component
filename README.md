# QR-Code-Component
### Project Description
This project is a simple QR code component project from Frontend Mentor. This project consists of a centered div with a QR code, a heading and some text in the website which is responsive so that when we make the window smaller, the div moves along with it. 

### What I learnt
- Flexbox Positioning: I managed to make the div in the center

### What was used?
- HTML5
- CSS3
- Flexbox
